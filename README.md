# Task 1: Iris Dataset Analysis

## Objective
To explore and visualize the Iris dataset to understand relationships between different features.

## Dataset Description
The Iris dataset contains measurements of iris flowers with the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Setosa, Versicolor, Virginica)

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
- Loaded dataset using seaborn
- Explored dataset structure using:
  - `.shape`
  - `.columns`
  - `.head()`
- Created visualizations:
  - Scatter Plot
  - Histogram
  - Box Plot

## Key Insights
- Different species show clear separation
- Petal length is a strong distinguishing feature
- Visualizations help identify patterns and outliers

# Task 4: Insurance Cost Prediction (Regression)

## Objective
To predict medical insurance charges based on personal attributes.

## Dataset
Medical Cost Personal Dataset

## Tools & Libraries
- Python
- Pandas
- Seaborn
- Scikit-learn

## Steps Performed
- Loaded and explored dataset
- Performed data visualization:
  - Age vs Charges
  - BMI vs Charges
  - Smoking vs Charges
- Converted categorical variables using encoding
- Split dataset into training and testing sets
- Applied Linear Regression model

## Results
- Model evaluated using:
  - MAE (Mean Absolute Error)
  - RMSE (Root Mean Squared Error)

## Key Insights
- Smoking significantly increases insurance charges
- Age and BMI also impact costs
- Linear Regression provides reasonable predictions

# Task 2: Credit Risk Prediction (Classification)

## Objective
To predict whether a loan application will be approved or not.

## Dataset
Loan Prediction Dataset

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
- Loaded dataset and cleaned missing values
- Dropped unnecessary columns (Loan_ID)
- Performed data visualization:
  - Loan Amount Distribution
  - Education vs Loan Status
  - Income vs Loan Amount
- Encoded categorical variables
- Split dataset into training and testing sets
- Applied Logistic Regression model

## Results
- Accuracy: **~76%**
- Evaluated using:
  - Accuracy Score
  - Confusion Matrix

## Key Insights
- Income and credit history strongly influence loan approval
- Model predicts approved loans well
- Some misclassification exists for rejected applications
- Model performance can be improved using advanced algorithms
