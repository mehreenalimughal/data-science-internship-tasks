## 🔹 Task 1: Iris Dataset Analysis

### 📌 Objective
The objective of this task is to explore and visualize the Iris dataset to understand relationships between different features using Python libraries.

---

### 📊 Dataset Description
The Iris dataset contains measurements of iris flowers. It includes the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Setosa, Versicolor, Virginica)

---

### ⚙️ Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

### 🔍 Steps Performed

1. Loaded dataset using seaborn
2. Explored dataset structure using:
   - `.shape`
   - `.columns`
   - `.head()`
3. Performed data visualization:
   - Scatter Plot (relationship between variables)
   - Histogram (distribution of values)
   - Box Plot (outliers and spread)

---

### 📈 Key Insights

- Different species have distinct characteristics
- Petal length is a strong distinguishing feature
- Data visualization helps in understanding patterns clearly
